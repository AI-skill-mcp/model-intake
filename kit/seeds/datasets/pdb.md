# Protein Data Bank (PDB)

全球蛋白/核酸/复合物实验三维结构权威数据库，由 wwPDB 联盟维护。

**官方来源**：https://www.rcsb.org/  
**文献**：10.1093/nar/gkab1023

---

## 基本信息

| 字段 | 值 |
|------|-----|
| `dataset_id` | pdb |
| `name` | Protein Data Bank (PDB) |
| `dataset_type` | reference |
| `description` | 全球蛋白/核酸/复合物实验三维结构权威数据库，由 wwPDB 联盟维护。 |
| `size_description` | >200,000 实验结构（持续增长） |
| `modalities` | [protein_structure, nucleic_acid_structure] |
| `organizations` | [wwPDB, RCSB, PDBe, PDBj, BMRB] |
| `url` | https://www.rcsb.org/ |
| `paper_doi` | 10.1093/nar/gkab1023 |
| `license_note` | 开放访问，引用结构发布者 |

## 验证说明

本条目为可公开核验的生物数据资源，须同时提供官方 `url` 与 `paper_doi`。
